* Directus Template CLI

A CLI tool for Directus templates

This fork is mainly dedicated to CMS

** Running the template CLI tool
Firstly, do the build after changes

#+begin_src 
  npm run build
#+end_src

Nextup, run the tool
#+begin_src
  bin/run.cmd apply
#+end_src
